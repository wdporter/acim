This is a catena aurea (golden chain) for A Course in Miracles.

The course text is supplied in alternative renderings which substitute pronouns.

I have provided my own summary for each paragraph.

I have also provided excerpts from well known course commentators.

I very much welcome correspondence with suggestions for improvements or additions.

The site is built with static site generator Jekyll and the sources are in github https://github.com/W-David-Porter/acim.

If you don't know what github is, you can email me instead.

Help with CSS and site design would be appreciated also.
